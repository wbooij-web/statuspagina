Beheer via Google Sheets

Statussen bijwerken zonder GitHub-token

ICT-collega's beheren de status in een gedeelde Google Sheet. De medewerkerspagina leest die sheet automatisch uit.

Bekijk monitor

Werkwijze

Configuratie controleren...

  1. Maak een Google Sheet met de kolommen: id, naam, status, opmerking, zichtbaar.
  2. Gebruik bij status alleen groen, oranje of rood.
  3. Gebruik bij zichtbaar ja of nee.
  4. Kies in Google Sheets: Bestand > Delen > Publiceren op internet.
  5. Kies het tabblad Statussen en publiceer als CSV.
  6. Kopieer de CSV-link naar sheet-config.json in GitHub.

Configuratie

Vervang de lege waarde door de gepubliceerde CSV-link.

{
  "csvUrl": "https://docs.google.com/spreadsheets/d/e/.../pub?gid=0&single=true&output=csv"
}

Na het opslaan in GitHub kan GitHub Pages heel kort nodig hebben om de wijziging zichtbaar te maken. Google Sheets zelf kan ook enkele minuten cachen.
